# 🏥 Patient Dashboard

A responsive healthcare dashboard built using HTML, CSS, and JavaScript that integrates with a live API to display patient data.

---

## 🚀 Features

- 📡 API Integration (Basic Auth)
- 📊 Dynamic Chart (Chart.js)
- 🧠 Data Binding (Real-time UI updates)
- 📋 Diagnostic List
- 🧪 Lab Results
- ❤️ Vital Signs Monitoring

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js

---

## 🔐 API Used

Coalition Technologies Patient Data API
